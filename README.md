# EasyAutoScript

一些简易的自动化脚本,用于创建主机后的简单设置。

## Main  
通用于 Linux 操作系统的脚本
## Debian / Ubuntu
- **通用于桌面和服务器的脚本**
- **desktop**
用于 Ubuntu / Debian 桌面环境(GNOME3)的脚本
- **server**
用于 Debian 服务器端的脚本
- **sources**
用于切换 apt 源为国内服务器
通常为[清华大学开源软件镜像站](https://mirrors.tuna.tsinghua.edu.cn/)或[阿里巴巴开源镜像站]  (https:opsx.alibaba.com/mirror)
